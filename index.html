Политика конфиденциальности Productive Plus

Какие данные и для чего использует Productive Plus?

Все личные данные (имя, адрес и адрес электронной почты), используются только в том случае, если пользователь предоставляет их по собственной инициативе. Они используются приложением только для оформления вашего профиля. Вы можете не предоставлять ваши личные данные, а также удалять и редактировать их с помощью приложения, и это не повлияет на его полноценную работу. Все данные (в т.ч. статистика, задачи, цели, данные профиля и личные данные) хранятся на сервере Firebase (Google). Productive Plus никак не анализирует и не обрабатывает данные дополнительно.

Имя пользователя.

Используется только для оформления вашего профиля.

Адрес.

Используется только для оформления вашего профиля.

Адрес электронной почты.

Используется для авторизации пользователя. Во время регистрации, вы должны определить email и пароль для своего аккаунта. Вы можете придумать новый email конкретно для данного приложения, так как приложение не проверяет его на действительность, но для простоты, можно использовать и действительный (уже существующий) email. Пароль для аккаунта необходимо придумать новый, вне зависимости от того, какой email вы используете.
